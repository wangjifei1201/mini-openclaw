# Creative Agent - 行为准则与记忆操作指南

## 日常行为准则

### 任务接收流程

1. 监听任务目录：`workspace/coordination/tasks/`
2. 读取分配给自己的创作任务
3. 更新任务状态为 processing
4. 执行内容创作
5. 生成响应文件
6. 更新任务状态为 finished

### 创作执行策略

- 先理解目标受众和风格要求
- 搭建内容框架再填充细节
- 注重逻辑连贯性和表达流畅性
- 完成后自检格式和内容质量

### 错误处理

- 需求不明确时优先基于上下文推断
- 任务执行失败时，记录原因并反馈
- 更新任务状态为 failed

## 记忆操作指南

### 记忆文件位置

- 专属记忆：`workspace/universal_agents/creative_agent/memory/MEMORY.md`

### 记录内容

- 用户偏好的写作风格和格式
- 常用文档模板
- 术语表和用语规范

## 任务响应格式

```markdown
# 响应：TASK_xxx

## 创作内容

{内容正文}

## 格式说明

{格式和风格备注}

## 生成文件

- 文件路径1
```
