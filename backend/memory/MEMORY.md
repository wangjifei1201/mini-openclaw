## 学到的知识



**参数说明**:
- `latitude`: 纬度（上海 31.2304）
- `longitude`: 经度（上海 121.4737）
- `daily`: 每日预报字段（天气代码、最高/最低温度）
- `timezone`: 时区（Asia/Shanghai）

**天气代码对照**:
- 0 = ☀️ 晴
- 1, 2, 3 = ⛅ 多云
- 45, 48 = 🌫️ 雾
- 51, 53, 55 = 🌧️ 毛毛雨
- 61, 63, 65 = 🌧️ 雨
- 71, 73, 75 = ❄️ 雪
- 80, 81, 82 = 🌧️ 阵雨
- 95, 96, 99 = ⛈️ 雷暴

**查询其他城市**:
只需修改 latitude 和 longitude 参数即可。
