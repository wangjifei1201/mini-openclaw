# 北京慧点科技出差借款报销助手

## 任务目标
- 本 Skill 用于：根据北京慧点科技有限公司财务借款报销管理制度，分析出差借款与报销场景，提供所需的资料清单、时限要求和注意事项
- 能力包含：
  - 分析出差和借款场景（国内差旅/个人借款/招待费等）
  - 提供所需的报销资料清单
  - 说明报销条件和限制（时限、标准、发票要求）
  - 生成报销清单模板
- 触发条件：用户提及出差报销、借款申请、报销时限、发票要求等

## 前置准备
- 了解公司财务借款报销管理制度
- 确认借款用途和还款期限
- 收集出差相关票据和凭证
- 准备招商银行账户信息

## 操作步骤

### 标准流程

1. **收集基本信息**
   - 询问用户需求类型：
     - 出差差旅报销
     - 个人借款申请
     - 招待费报销
     - 会议/培训费用报销
   - 获取具体信息：
     - 费用类型（交通、住宿、餐饮、招待等）
     - 发生时间、地点、事由
     - 金额及付款方式

2. **分析所需资料和时限**
   - 根据费用类型，参考 [references/expense-policy-template.md](references/expense-policy-template.md) 中的管理制度
   - 列出所需的报销资料清单，参考 [references/required-documents.md](references/required-documents.md)
   - 明确报销时限：
     - 经济业务发生后30个工作日内提交纸质报销单
     - 最迟不超过3个月（发票有效期）
     - 按季度生成报销单号（季度末、次月末提交）

3. **说明报销条件和注意事项**
   - 根据费用类型，说明报销标准：
     - 差旅费：飞机经济舱、高铁二等座等具体标准
     - 招待费：人均200-600元，禁止烟酒娱乐礼品
     - 餐饮补贴：按出差天数计算
   - 发票要求：
     - 发票抬头必须是"北京慧点科技有限公司"
     - 发票内容与实际业务相符（三流合一）
     - 通过国家税务总局发票查验平台验证
   - 提示特殊情况的补充材料要求

4. **生成报销清单**
   - 根据用户实际费用情况，生成详细的报销清单
   - 使用 [assets/reimbursement-form-template/domestic-travel-form.md](assets/reimbursement-form-template/domestic-travel-form.md) 生成报销表

### 可选分支

- **当涉及个人借款时**：参考借款管理章节，重点说明：
  - 借款原则："一单一清"、"前帐不清，后账不借"
  - 借款用途限制：中标服务费、标书款、投标保证金、诉讼费、律师费、会议费、培训费、房租、活动费、场地费、项目费用
  - 还款期限：本月借款需在下月月底前提交报销冲抵
  - 超过3个月未还：停止该员工及部门的所有借款业务

- **当涉及出差差旅时**：参考差旅费报销章节，重点说明：
  - 交通标准：飞机经济舱、高铁二等座、火车硬卧、轮船三等舱
  - 住宿标准：一线城市450-550元，二线及以下300-400元
  - 餐饮补贴：出差期间吃住行费用一起填报
  - 核酸检测费用：出差期间同差旅费用一起报销；工作常驻地需写明原因

- **当涉及招待费时**：参考招待费管理章节，重点说明：
  - 人均标准：重要客户600元，一般客户400元，公务招待200-400元
  - 禁止项目：礼品、烟、娱乐、旅游、鱼翅燕窝、野生动物
  - 单张发票不超过2000元（集中采购的酒水茶叶纪念品除外）
  - 销售人员需部门总监签字确认，注明招待人数

- **当涉及会议/培训费时**：参考会议培训费章节，重点说明：
  - 事前审批：组织前填写申请表，附相关通知
  - 通知要求：时间、地点、参加人员、内容、经费及安排
  - 劳保/福利用品：需附销货单位加盖公章的明细清单

## 注意事项

- 仅在需要时读取参考文档，保持上下文简洁
- 发票必须在国家税务总局全国增值税发票查验平台验证（https://inv-veri.chinatax.gov.cn/）
- 报销对私付款统一为招商银行，收款人必须与申请人一致
- 上年度发票使用截止到当年2月底（以电子系统单据编号为准）
- 每年12月1日前未走完流程的报销单将停止付款结算，次年1月20日开启
- 劳务费只能以报销形式领取，需附有领取人签字的身份证复印件，直接转款至银行卡

## 使用示例

### 示例1：国内差旅报销资料和时限查询
**功能说明**：提供国内出差报销所需的完整资料清单和时限要求
**执行方式**：智能体根据管理制度生成清单
**关键要点**：
- 询问出差时间、地点、事由
- 说明报销时限：经济业务发生后30个工作日内提交，最迟3个月
- 说明季度报销单号生成时间：12-2月业务2月28日前生成单号，3月31日前提交纸质单据
- 列出基础单据：交通票据、住宿发票、行程单
- 说明发票抬头必须是"北京慧点科技有限公司"

**功能说明**：提供个人借款的申请条件、用途限制和还款要求
**执行方式**：智能体根据借款管理制度提供指导
**关键要点**：
- 借款原则："一单身清"、"前帐不清，后账不借"
- 借款用途：中标服务费、标书款、投标保证金、诉讼费、律师费、会议费、培训费、房租、活动费、场地费、项目费用
- 禁止备用金形式借款
- 还款期限：本月借款需在下月月底前提交报销冲抵
- 超过3个月未还将停止该员工及部门的所有借款业务

### 示例3：招待费报销指导

- 提供招商银行账户信息
- 注意报销时限和发票要求
