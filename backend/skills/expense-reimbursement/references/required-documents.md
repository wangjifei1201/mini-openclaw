## 概览
本文档提供北京慧点科技有限公司报销所需的完整资料清单，根据业务类型选择对应的资料。所有单据需真实、合法、有效，发票需在全国增值税发票查验平台验证。

## 基础单据（所有场景必备）

### 1. 报销申请单
**格式要求**：
```
报销申请单格式：
{
  "申请人": "姓名",
  "部门": "部门名称",
  "职位": "职务",
  "报销日期": "YYYY-MM-DD",
  "费用类型": "差旅费/招待费/会议费/培训费/其他费用",
  "经济业务发生日期": "YYYY-MM-DD",
  "报销总金额": "元",
  "招商银行账号": "账号信息",
  "审批人": "姓名及职务",
  "审批日期": "YYYY-MM-DD"
}
```

**验证规则**：
- 必须在经济业务发生后30个工作日内提交纸质报销单
- 最迟不可超过经济业务发生日期3个月
- 需按季度规定时间生成报销单号并提交纸质单据
- 审批流程完整（按公司授权框图执行审批）

### 2. 发票及查验结果
**发票要求**：
- 发票抬头必须是"北京慧点科技有限公司"
- 发票内容与实际业务相符（三流合一）
- 通过国家税务总局发票查验平台验证

**查验平台**：https://inv-veri.chinatax.gov.cn/

**格式要求**：

## 发票查验资料

### 查验平台
国家税务总局全国增值税发票查验平台：https://inv-veri.chinatax.gov.cn/

### 查验步骤
1. 进入查验平台
2. 填写发票信息：
   - 发票代码
   - 发票号码
   - 开票日期
   - 开具金额（不含税）
   - 校验码（图片中的数字）
3. 点击查验
4. 查验通过后，点击打印
5. 将查验结果打印作为附件附在发票后

### 必须查验的发票类型
- 增值税专用发票
- 电子增值税普通发票
- 电子发票

### 查验结果格式
```
发票查验结果示例：
查验日期：2024年01月15日
发票代码：1100174320
发票号码：12345678
开票日期：2024-01-10
开具金额（不含税）：900.00元
查验结果：查验通过
```

## 注意事项

### 单据整理要求
1. 按费用类型分类整理
2. 按时间顺序排列
3. 附上对应的说明材料
4. 使用回形针或文件夹固定，避免订书钉损坏票据

### 单据保存要求
1. 原件必须完整、清晰
2. 避免涂改、折叠严重
3. 发票联次完整（增值税专用发票需发票联和抵扣联）
4. 电子发票需打印并附查验结果

### 单据有效性要求
1. 发票必须在有效期内（最迟不超过3个月）
2. 发票信息必须准确无误
3. 发票必须盖有发票专用章
4. 电子发票需打印并查验

### 发票抬头要求
所有发票的付款单位名称必须是"**北京慧点科技有限公司**"（火车票、电子客票行程单、轮船票等必须填写个人身份证号）。

### 发票项目要求
- 发票项目应按照经济业务实际情况开具
- 不得开具"办公用品"、"耗材"、"礼品"等笼统名称
- 会议费只能开具：会务费、会议注册费、会议费
- 招待费只能开具：餐饮费
- 培训费只能开具：培训费

### 发票备注要求
- **住宿费**：备注栏必须填写住宿天数和人数
- **会议费**：备注栏必须填写会议地点
- **培训费**：备注栏必须填写培训地点和培训天数
- **建筑服务**：备注栏必须填写项目名称、发生地点

### 报销时限要求
1. **及时性**：需在经济业务发生后30个工作日内提交纸质报销单到财务审核
2. **最长期限**：最迟不可超过经济业务发生日期3个月
3. **季度规定**：
   - 12-2月业务：2月28日前生成单号，3月31日前提交纸质单据
   - 3-5月业务：5月31日前生成单号，6月30日前提交纸质单据
   - 6-8月业务：8月31日前生成单号，9月30日前提交纸质单据
   - 9-11月业务：11月30日前生成单号，12月31日前提交纸质单据
4. **停止结算**：每年12月1日前未走完流程的报销单将停止付款结算，次年1月20日开启

### 付款信息要求
1. 报销对私付款统一为招商银行付款
2. 收款人必须和申请人一致（对公付款除外）
3. 申请人所提供的招行卡号必须与申请人及收款人信息所对应
4. 所有报销人员务必每次报销都要在电子报销单中填写银行账号及信息

### 常见错误
1. ❌ 发票抬头填写错误或非公司全称
2. ❌ 发票项目填写笼统名称（办公用品、礼品等）
3. ❌ 发票备注信息缺失（住宿天数人数、会议地点等）
3. ❌ 发票未在全国增值税发票查验平台验证
4. ❌ 发票日期不在有效期内
5. ❌ 票据丢失未提供证明材料
6. ❌ 缺少审批文件
7. ❌ 单据混乱、未分类整理
8. ❌ 招待费购买礼品、烟、娱乐等禁止项目
9. ❌ 超期报销未审批
