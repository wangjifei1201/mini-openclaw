# 天气查询技能

## 功能说明

这个技能用于查询指定城市的实时天气信息。

## 使用步骤

1. 使用 `fetch_url` 工具访问 wttr.in 天气服务
2. 解析返回的天气数据
3. 以友好的格式回复用户

## 具体操作

### 方法一：使用 wttr.in（推荐）

wttr.in 是一个免费的命令行天气服务，支持中文城市名。

```
fetch_url(url="https://wttr.in/{城市名}?format=3")
```

例如查询北京天气：
```
fetch_url(url="https://wttr.in/Beijing?format=3")
```

### 方法二：获取详细天气

如果需要更详细的天气信息：
```
fetch_url(url="https://wttr.in/{城市名}?lang=zh")
```

## 输出格式

请将获取到的天气信息整理成以下格式回复用户：

```
【{城市名}天气】
- 天气状况：{天气描述}
- 当前温度：{温度}
- 体感温度：{体感温度}
- 湿度：{湿度}
- 风速：{风速}
```

## 注意事项

- 城市名可以使用中文拼音或英文
- 如果用户提供中文城市名，转换为拼音使用
- 如果获取失败，提示用户检查城市名是否正确
- 当请求该服务为获取到预期结果是，直接给出友好提示即可，注明是接口问题。
